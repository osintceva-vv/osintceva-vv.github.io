```html Симулятор подбрасывания монеты

Симулятор подбрасывания монеты

Орел
Решка

``` **Изменения в коде:** * **Удалена анимация вращения:** Удален CSS-стиль `animation` для элемента `.coin`, так как нам не нужна анимация. * **Удален `setTimeout`:** Удален `setTimeout`, который задерживал вывод результата. Теперь результат выводится сразу же после завершения цикла. * **Упрощен цикл:** В цикле `for` оставлен только код для генерации случайного числа и увеличения счетчиков. **Как работает:** 1. Пользователь вводит количество бросков в поле "Количество бросков". 2. Нажимает на кнопку "Подбросить". 3. Цикл `for` выполняет подбрасывание монеты заданное количество раз. 4. Для каждого броска генерируется случайное число, а затем увеличивается счетчик "Орлов" или "Решек" в зависимости от результата. 5. После завершения всех бросков, результат выводится сразу же в формате: "Результат: \nОрел: [количество], Решка: [количество]". Теперь симулятор подбрасывает монету нужное количество раз и выводит результат мгновенно, без анимации.
